## [0.3.2] - 2026-03-31

### Added

- Initial public release of `@software-mansion/argent` MCP package.
- Tools server (`@argent/tool-server`) HTTP API for iOS simulator control.
- Registry (`@argent/registry`) with dependency-aware service lifecycle and blueprint system.
- MCP bridge (`packages/mcp`) exposing all tools to Claude and Cursor via Model Context Protocol.
- Web UI (`@argent/ui`) for simulator control and Metro debugging.
- Skill files (`@argent/skills`) for AI agent guidance.
- `simulator-server` native binary for device I/O (touch, keys, buttons, rotation, MJPEG stream, screenshots, recording).
